# Sign up / Login API by

## Integrated by Node.js, Express.js, MongoDB

## List Of api:

###### Signup

> HOSTNAME/signup

- Method `POST`
- Body `name`,`email`,`pass`

###### Login

> HOSTNAME/login

- Method `GET`
- Body `email`,`pass`

###### get profile

> HOSTNAME/

- Method `GET`
- headers `token`
